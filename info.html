About This Solution

The Cattle Disease Detector is an AI-powered solution built using a Convolutional Neural Network (CNN) model to identify common diseases in cattle, including:

  • Lumpy Skin Disease
  • Mastitis
  • Foot and Mouth Disease (FMD)
  • Healthy

The solution helps farmers, veterinarians, and agricultural institutions detect diseases early and take prompt action. The model was trained using Artificiall intelligence and tested on a well-structured image dataset.
